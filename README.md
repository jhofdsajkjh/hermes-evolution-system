# Hermes Evolution System

## 神级基因 + 顶级代码

Hermes Agent 自我演进系统，目标是打造"神级基因"（最优质的代码模版）和"顶级代码"（经过验证的最佳实践）。

### 核心组件

- `genes/` - 代码基因库（可复用的最佳实践）
- `evolve.py` - 演进主程序
- `闭环.md` - 演进闭环文档

### 演进流程

1. 学习（从优秀源码提取基因）
2. 验证（通过测试确保基因有效）
3. 沉淀（存入知识池）
4. 复利（下一代基于前代积累）
5. 进化（基因突变和选择）
